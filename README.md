## 🙋 Frequently Asked Questions

What should I do if the software freezes?
Restart the application. If the problem continues, delete the local cache folder located in your AppData directory and restart the program.

Can I sync this with my mobile wallet?
The software works independently. You can copy addresses from your mobile wallet and paste them into the desktop interface, but the desktop tool maintains a separate balance record.

Why does the network show as disconnected?
Check your firewall settings. Sometimes Windows Defender blocks new software from web access. Allow the application through your firewall to fix this connection error.

Are my transactions private?
The software interacts with the public blockchain. While the tool manages your keys locally, the activity remains public on the ledger. Be mindful of which addresses you interact with.

Is there a fee for using this tool?
The software operates based on standard network gas fees. The tool does not charge extra commissions for standard transactions.
